# LAS2023 Official Website

정적 HTML/CSS/JavaScript 기반의 LAS2023 공식 홈페이지 초안입니다.

## 실행 방법
1. VS Code에서 이 폴더를 엽니다.
2. `index.html`을 우클릭합니다.
3. `Open with Live Server`를 실행합니다.

## 매달 상영일정 이미지 교체
아래 파일만 같은 이름으로 덮어쓰면 됩니다.

`images/cinema/this-month.jpg`

## 메인 대표 사진 교체
`images/home/hero.jpg`

## About 사진 교체
`images/about/about.jpg`

## 컬렉션 데이터 수정
`data/collection.json` 파일에 항목을 추가하면 Collection 페이지 검색 결과에 반영됩니다.

## 주요 파일
- `index.html` : 메인 페이지
- `css/style.css` : 전체 디자인
- `js/script.js` : 컬렉션 검색 기능
- `pages/` : 하위 페이지
- `images/` : 홈페이지 이미지
- `data/collection.json` : 컬렉션 샘플 데이터
